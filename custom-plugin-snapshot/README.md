# Custom Plugin Snapshot

This folder contains the current customized Bragi Canvas build that is being used locally.

Included:

- `manifest.json`
- `main.js`
- `styles.css`

Not included:

- `data.json`
- API keys
- canvas files
- generated images or videos
- vault content

This snapshot is provided so maintainers can inspect the full customized plugin behavior while individual features are split back into source-level pull requests.
